1.Introduction
HI Professor，The Custom Noise Texture Generator（User_Input.py） is a Python application designed for artists who need various types of noise textures for displacement mapping, special effects, and other creative operations. This tool allows you to generate seamless noise textures like volcanoes, grasslands, and snow surfaces. Each texture type comes with adjustable parameters, enabling you to create unique and customized textures to suit your artistic needs.

2. Install Required Libraries
The required libraries are written in the requirements.txt file(\assignment\Assignment_3\requirements.txt)
pip install -r requirements.txt
